# Krishnam Raju - Professional Developer Portfolio

A modern, highly interactive, and visually stunning Next.js portfolio website built with Vanilla CSS and Framer Motion.

## Features

- **Rich Visual Aesthetics**: Harmonious dark/light mode palettes, glassmorphism, dynamic gradients, ambient light blobs, and particle effects.
- **Scroll Reveal Animations**: Smooth scroll-triggered animations via Framer Motion.
- **Interactive 3D Cards**: Hover-controlled 3D tilt effects using vanilla React state.
- **Interactive Contact Form**: Client-validated message form with immediate feedback.
- **Complete Sections**: Hero, About, Skills, Projects, Experience, Salesforce PD1, Education, Certifications, Strengths, Blog, and Contact.

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd krishnam-raju-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it.
